# Sprint 1 - Project Foundation

**Sprint Goal**

Build the foundation of Venture Herd Manager by creating a working ASP.NET Core Web API, establishing the project architecture, and implementing basic animal management.

---

## User Story

As a herd manager,

I want to create and view animals,

So that I can begin building a complete digital herd management system.

---

## Objectives

- Build the project foundation
- Learn Git and GitHub
- Learn ASP.NET Core Web API
- Learn Controllers
- Learn Services
- Build the Animal model
- Build a working REST API
- Prepare for Azure SQL
